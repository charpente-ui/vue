## Introduction

Charpente UI is a headless CSS Vue component library.

## Installing

```shell
npm install @charpente-ui/vue
```

## Usage

```vue
import { CButton } from '@charpente-ui/vue';

<template>
    <CButton>Click me</CButton>
</template>
```
